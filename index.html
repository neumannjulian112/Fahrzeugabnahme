SO FUNKTIONIERT'S

Mobile Abnahme in 4 Schritten

  1. Datei laden — Excel-Protokoll aus den Dateien öffnen.
  2. Fahrzeuge wählen — Hauptfahrzeug + bis zu 3 weitere für gemeinsame Bearbeitung.
  3. Position prüfen — Status setzen, Mangel beschreiben, mit „Weiter".
  4. Übernehmen — gleichen Befund auf weitere Fahrzeuge mit einem Tipp übernehmen.

Klick auf „Erledigt" oder „Mangel" gibt einen kurzen Bestätigungston. Im Header oben rechts kannst du Töne deaktivieren.
