# Abnahme-App v3.2 — Feuerwehr Frankfurt

Mobile Web-App für Fahrzeug-Abnahmeprotokolle.

## v3.2: Bugfix

- "Zusätzliche Mängel"-Screen wurde nicht aktiv geschaltet (Eintrag nicht möglich)

## v3.1: Lesbarkeit im Tageslicht

Schriftfarben für hellere Umgebungen optimiert.

## v3.0-Funktionen

- App-Icons mit Adler-Logo
- Foto-Anhänge an Mangel-Positionen (max. 3 pro Position)
- Freie Mängel pro Tabellenblatt+Fahrzeug
- Export mit Fotos als ZIP-Bundle
- PWA-Polish

## Installation auf GitHub Pages

1. Ordner-Inhalt in **public** Repository hochladen
2. Settings → Pages → Source `main` / `/ (root)` → Save
3. Nach 1-2 Min unter `https://DEINNAME.github.io/REPO/`
